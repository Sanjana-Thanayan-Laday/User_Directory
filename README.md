# User Directory

This is a simple **User Directory** web application built with **HTML, CSS, and JavaScript**. The application displays a list of users and allows basic interaction using front-end code only.

## Project Structure

- **index.html** — Main web page that shows the user directory interface.
- **style.css** — Styles and layout for the user directory UI.
- **script.js** — JavaScript logic to control dynamic behavior (fetching data, rendering users, filtering, etc.).

## Features

- Responsive layout for displaying users.
- Client-side interaction using JavaScript.
- Clean UI styled with CSS.
